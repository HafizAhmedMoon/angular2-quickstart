# angular2-quickstart
Just type `npm start` to say Hello Angular2

## Usage


```bash
git clone --depth 1 https://github.com/hafizahmedattari/angular2-quickstart.git
cd angular2-quickstart

# install the project's dependencies
npm install

# watch and serve your files
npm start

# build
npm run build
```
